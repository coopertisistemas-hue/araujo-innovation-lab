# Fluxos de Trabalho (Workflows)

## Propósito
Padronizar a execução de tarefas recorrentes, garantindo qualidade, conformidade institucional e eficiência.

## Estrutura
Documentação de passos sequenciais, responsabilidades e critérios de sucesso para processos como Aprovação Institucional e Fluxo de Branding.

## Uso
Deve ser consultado antes do início de qualquer nova atividade padrão para garantir que os processos estão sendo seguidos.

## Integração
Orquestra a colaboração entre `agents`, `subagents` e stakeholders humanos.
