# Documentação Institucional

## Propósito
Este diretório centraliza toda a documentação oficial, estratégica e pedagógica do Araujo Innovation Lab. É a fonte da verdade para diretrizes, relatórios e registros formais.

## Estrutura
- `executive/`: Documentos de visão macro e apresentações para liderança.
- `internal/`: Planejamentos, atas de reuniões e materiais de uso restrito.
- `public/`: Conteúdos destinados à comunidade externa e parceiros.
- `meetings/`: Registros históricos de decisões e alinhamentos.
- `governance/`: Manuais de processos e regras de conformidade.
- `reports/`: Relatórios de impacto e indicadores de desempenho.

## Uso
Todos os documentos devem seguir os templates institucionais localizados em `/templates`. O tom deve ser sempre executivo e profissional.

## Integração
Conecta-se com o sistema de `memory` para registro de decisões e com `workflows` para os processos de aprovação documental.
