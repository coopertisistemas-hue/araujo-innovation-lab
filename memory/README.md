# Sistema de Memória

## Propósito
Preservar o contexto estratégico, as decisões históricas e o roadmap de longo prazo, evitando a perda de conhecimento entre ciclos anuais.

## Estrutura
- `context/`: Contexto atual do projeto e ambiente.
- `decisions/`: Registro de porquês e decisões críticas.
- `roadmap/`: Planejamento de expansão futura.

## Uso
Consultado para entender o histórico de decisões e para planejar os próximos passos estratégicos.

## Integração
Informa a evolução de todos os outros diretórios, servindo como o "cérebro" do repositório.
