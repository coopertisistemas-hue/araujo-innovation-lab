# Matriz de Skills

## Propósito
Define as capacidades técnicas e competências que o ecossistema é capaz de executar. Serve como um inventário de conhecimentos aplicados.

## Estrutura
Documentação de competências como Gestão de Reputação Digital, Planejamento Educacional e Operações de Social Media.

## Uso
Referência para o desenvolvimento curricular e para a atribuição de tarefas aos subagentes.

## Integração
Alimenta os `workflows` e define os requisitos para os projetos estudantis em `projects`.
