# Agentes Institucionais

## Propósito
Concentra as definições e instruções para os Agentes de IA que representam funções estratégicas no ecossistema. Estes agentes garantem a consistência da visão institucional em diferentes áreas.

## Estrutura
Contém arquivos `.md` que definem o perfil, tom de voz e responsabilidades de cada cargo institucional (ex: AI Specialist, Branding Director).

## Uso
Utilizado para configurar sistemas de IA (como ChatGPT ou agentes locais) para que atuem conforme a persona institucional definida.

## Integração
Estes agentes orientam a execução dos `workflows` e a criação de conteúdos para as `social-media`.
